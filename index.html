⚙️ Configurar Firebase

Para sincronizar entre dos teléfonos necesitás configurar Firebase (gratuito). Seguí estos pasos una sola vez:

1 Crear proyecto en Firebase

Entrá a console.firebase.google.com → "Crear proyecto" → poné un nombre (ej: "casa-finanzas") → desactivá Google Analytics → Crear.

2 Crear base de datos Firestore

En el menú izquierdo → Firestore Database → "Crear base de datos" → elegí modo de prueba → elegí región más cercana → Listo.

3 Obtener configuración

Rueda de configuración (arriba izq.) → Configuración del proyecto → desplazate hasta "Tus apps" → "</>" (Web) → registrá la app → copiá el objeto firebaseConfig.
